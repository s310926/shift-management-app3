shift-management-app

これはJavaとServlet/JspとDBを使って作ったシフト管理の練習アプリです
ログインしてシフト登録したり、一覧で確認できるようになっています

＜このアプリでできること＞
-ログイン画面からユーザー認証、登録
-シフト希望の追加
-シフト一覧の表示

＜使用した技術＞

-Java(Servlet/JSP)
-HTML/CSS

＃作った理由
前職で２交代のシフト制での勤務をしており、その際のシフト希望やシフト調整など
紙ベースですごく非効率だと思ったので作成してみました
管理者権限でシフト作成する際に名前とIDを表示させるようにしたのは
人によって能力差があってそれを考慮してシフトを組む必要があるため、
管理者でも名前をみて分かりやすくシフトが組めるようにしました
