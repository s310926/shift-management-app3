# シフト管理アプリ

## プロジェクト概要

このプロジェクトはシフト管理を行うためのWebアプリケーションです。
ユーザーがカレンダー形式で希望シフトを入力し、登録することができます。
管理者画面では登録されたユーザーのシフトを閲覧、シフトの調整、管理をすることができます。

## 主な機能
- ログイン画面からユーザー認証、新規登録
- ユーザーIDの番号で管理者と一般ユーザーの管理
- シフト種別選択
- 時間帯選択
- 前月、来月への切り替え機能
- 入力内容の登録処理

## 技術構成

- Java(Servlet/JSP)
- HTML/CSS
- JavaScript jQuery

## 使い方

1. ログイン画面にて新規登録済の方はログインを、まだの方は新規ログイン画面へ遷移します。
2. ログインIDは、5000番以上が一般ユーザー、5000番以下が管理者ユーザーです。
3. 一般ユーザーではシフトの確認とシフトの登録ができます。
4. シフト入力画面を開くと当月のカレンダーが表示されます。
5. 各日付のセルでシフトの種別を選択します。
6. 時間指定を選んだ場合は、時間帯を選択します。
7. 入力完了したら登録するボタンを押します。
8. 前月や来月ボタンで表示月を切り替えることができます。
9. 管理者画面では、一般ユーザーのシフトの希望情報を見ることができます。
10. 編集するボタンを押すと、シフトの変更ができます。

## 制作した理由
前職で２交代のシフト制での勤務をしており、その際のシフト希望やシフト調整など
紙ベースですごく非効率だと思ったので作成してみました。
管理者権限でシフト作成する際に名前とIDを表示させるようにしたのは
人によって能力差があってそれを考慮してシフトを組む必要があるため、
管理者でも名前をみて分かりやすくシフトが組めるようにしました。
